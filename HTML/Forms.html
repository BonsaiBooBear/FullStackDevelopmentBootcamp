<!--
    # Forms:
         - Used to collect data from the user
         - Tags:
            - form: Wrapper tag for HTML Forms
            - input: Input field for user data
                - 'type' attribute
                    - text: Simple text input
                    - number: Number input
                    - email: Email input (Browser validation)
                    - password: Password
                    - date: Date input
                    - time: Time input
                    - datetime-local: Date + Time input
                    - file: File input
                    - radio: Single selection input
                    - checkbox: Multiple selection input
                    - button: Button input
                - 'name' attribute: Provides name to the input field
                - 'value' attribute: Default value for the input field
                - 'placeholder' attribute: Provide a hint
                - 'required' attribute: Make any field required (Browser validation)
                - 'disabled' attribute:  Disable and input field
            - button: Buttons
                - 'type' attribute
                    - submit: Used to submit a forms
                    - reset: Used to reset the form
                    - button: Random button in the form
            - select: Dropdown (Single selection)
                - option tag: Individual options
            - textarea: Lengthy text inputs
                - 'rows' attribute: Number of rows
                - 'cols' attribute: Number of columns


    # Additional Notes:
        - input type="hidden": Don't want to show a form, but want to process the values with other details
        - Before the button element was introduced, submit buttons were input types: <input type="button" value="Submit">
        - Self-closing tags will still be recognized by most browsers if they include the '/' but best practice is to not include them
            - Use <input> instead of <input />
            - Use <br> instead of <br />
