<!--
    # Tables:
        - Display data in rows and column format
        - Tags:
            - <table></table>: Wrapper tag for HTML Tables
            - <tr></tr>: Table rows
            - <th></th>: Table headers
            - <td></td>: Table data (cells)
        - Boilerplate syntax for tables:
            <table>
                <tr>
                    <th>Table Header 1</th>
                    <th>Table Header 2</th>
                </tr>
                <tr>
                    <td>Table Data 1</td>
                    <td>Table Data 2</td>
                </tr>
            </table>
        - Span (Merge)
            - colspan: Column Span
                - Data to be same across 'n' number of columns within a row
            - rowspan: Row Span
                - Data to be same across 'n' number of rows within same column
-->
