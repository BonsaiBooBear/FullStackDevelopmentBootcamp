     ## HTML(Hyper Text Markup Language)

            - Building blocks of HTML structure
            - Syntax: <tag1></tag1>
            - Types
                - Paired tags
                    - Opening and closing tags
                    - Examples: html, head, body, title, h1-h6, p, etc.
                - Self closing tags (Singleton tags)
                    - Opening tag
                    - Example: br, hr, img, input, etc. 
            - Definitions of tags
                - <html>: HTML document begins and closes with this tag
                - <head>: Includes details like title of the webpage and more
                - <title>: Define the title of the webpage 
                - <body>: Contains the actual content of the webpage
                - Headings
                    - <h1>: Heading 1 (Largest font size)
                    - <h2>: Heading 2
                    - <h3>: Heading 3 
                    - <h4>: Heading 4 
                    - <h4>: Heading 5 
                    - <h6>: Heading 6 (Smallest font size)
                - <p>: Paragraphs
                - <br>: Line break
                - <hr>: Horizontal line
        
        # Element
            - Tag along with content
                - <title>My first webpage</title>
        
        # Additonal pointers
             - HTML is case insensitive.
                    - <h1>Heading1</h1> and <H1>Heading1</H1> will give same O/P. But lowercase is a good practice.
                    - <!DOCTYPE html>: Tells browser that this document is HTML5 document (optional)
                    - Comments in HTML: Hide code/ Add notes between code
