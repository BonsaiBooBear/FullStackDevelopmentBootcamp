<!--
    # Semantic Tas:
        - Used to add meaning to the webpage structure
        - It may or may not have any visual representation
        - For search engines, for SEO purposes (Search Engine Optimization)
            - SEOs extract information from a webpage so it can be indexed and shown to users that enter search keywords.
        - Examples:
            - header: Should have Headings, Navbars, etc.
            - footer: Should have Sitemap
            - nav: Navigation links
            - main: Should have the main content of the webpage
            - section: Should have categorized information

    # Non-semantic tags:
        - Will not add any meaning to the webpage structure
        - Examples: div, span, etc.
-->
