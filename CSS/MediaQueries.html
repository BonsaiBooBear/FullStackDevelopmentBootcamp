    # Media Queries
        - Style rules for different screen sizes
        - Syntax
            @media mediaType and (condition) {
                //style to be applied to those devices which match the above condition
            }

        - mediaType
            - screen***: screen
            - print: Print
            - all: All media types

        - condition
            - max-width ***: Device width less than or equal to the given width
            - min-width ***: Device width greater than or equal to the given width
            - max-height: Device height less than or equal to the given height
            - min-height: Device height greater than or equal to the given height

        - Two approaches
            - Desktop-first approach:
                - Start with desktop, then add separate rules for mobile
            - Mobile-first approach:
                - Start with mobile, then add separate rules for desktop
        
        - Breakpoints
            - Laptops/Desktops: >= 922px ***
            - Tablets: > 480px and < 992px
            - Mobile: <= 480px ***

        - References:
            - https://getbootstrap.com/docs/5.3/layout/breakpoints
            - https://w3schools.com/howto/howto_css_media_query_breakpoints.asp
            - https://browserstack.com/guide/responsive-design-breakpoints
