    # Absolute vs Relative units
        - Absolute units:
            - Fixed size
            - Dimensions don't change based on parent element/viewport (browser) size.
            - Examples:
                - px: Pixels
                - pt: Points
        - Relative units:
            - Dynamic/Variable size
            - Dimensions change based on parent element/viewport (browser) size.
            - Examples:
                - %: Relative to parent element
                - vw: Relative to 'viewport width'
                    - 1vw = 1% of 'viewport width'
                - vh: Relative to 'viewport height'
                    - 1vh = 1% of 'viewport height'
                - em: 1 * Font size of the parent element
                    - 1em = Actual font size of parent element
                - rem: 1 * Font size of the root element (html)
                    - 1rem = Actual font size of root element
-->
