    # Specificity
        - Priority of a particular style over another style

        - Main Rule
            - Latest style rule applies
            - Inline > Internal, External

        - Priority Selector (rank from top to bottom)
            1. ID Selector (#id) - Highest
            2. Class Selector (.class)
            3. Element Selector (h1, p, div, etc.)
            4. Universal Selector (*) - Lowest Priority

        - Additional Note:
            - !important: This keyword overrides the other rules without specificity restrictions (Overuse is not a good practice)
-->
