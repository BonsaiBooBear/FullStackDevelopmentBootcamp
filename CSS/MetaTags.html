<!--
    # Meta Tags
        - Metadata: Data of data
        - Tag: <meta>
        - Always added within the head tag
        - Attributes:
            - charset: Character set (UTF-8 is a collection of all languages and symbols - used almost 100% of the time and will never be changed)
            - name
             - description: Description of webpage (Search Engine Optimizations)
             - keywords: Keywords related to the webpage (similar to hashtags)
             - author: Author of the webpage wrote the content
             - viewport: Set the viewport for different screen sizes
             - width=device-width: Setting viewport width to the device width (mobile ,tablet, etc.)
             - initial-scale=1.0: Default zoom
-->
