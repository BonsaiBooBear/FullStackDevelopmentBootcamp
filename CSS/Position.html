    # Position:
        - Set the position of the element on the webpage
        - Values:
            - static (default): Default position
            - relative ***: Relative to the original position
            - absolute ***: Relative to the nearest positioned ancestor/ webpage boundaries
                - If element is not nested, it will be positioned relative to the webpage.
                - If element IS nested, it will be positioned relative the the parent element, but only if the parent element has position property.
            - fixed: Fixed positioning
            - sticky: Relative, then fixed

        - Additional properties: Without the position property set to something other than 'static', these will not be recognzied.
            - top: Distance from top
            - right: Distance from right
            - bottom: Distance from bottom
            - left: Distance from left
