<!--
    # Flexbox ***
        - Organize elements in ONE direction
        - Properties
            - Container
                - display: flex ** (Element flexible in nature)
                - flex-direction ***: Set the direction of the flex container elements.
                    - row (default)
                - justify-content **: Align the child elements within the flex container (across main axis - default X Axis)
                    - start (default) Elements align at the start of the flex container
                    - end: Elements align to the end of the flex container
                    - center: Elements align in the center of the flex container
                    - space-around: Equal space on both sides of the elements (space in between will be 2* space)
                    - space-evenly: Equal space on both sides of the elements (space in between will be merged)
                    - space-between: Outside elements align to the start and end. Equal space will be between each of the remaining elements.
                - align-items **: Align the child elements within the flex container (through cross axis - default Y Axis)
                    - start (default): Elements align at the top of the flex container
                    - end: Elements align at the bottom of the flex container
                    - center: Elements align in the middle of the flex container
                - flex-wrap: Wrap the child elements if they exceed the parent flex container width
                    - no-wrap (default): Child elements will NOT wrap to the next row. Will have a horizontal scroll.
                    - wrap: Child elements WILL wrap to the next row if they cannot fit within the viewport.
                    - wrap-reverse: Child elements will wrap to the next row, but the order of elements will be reversed.
            - Item 
                - order: Set the order of individual items
                    - Min and Default (0): The lesser value, the higher order

        # Main and Cross axis
            - flex-direction: row (default)
                - Main axis: X Axis
                - Cross axis: Y Axis
            - flex-direction: column
                - Main axis: Y Axis
                - Cross axis: X Axis
-->
