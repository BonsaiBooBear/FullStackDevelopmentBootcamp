/*
  # Loops: Something to be executed on a repeated basis
    - Terms:
      - Iteration: Individual cycles
      - Condition: Determine to proceed or halt
    - Process:
      - In each iteration, condition will be checked
      - Only if condition is true, iteration will be processed
      - Else, loop will break
    - One similarity in all types of loops:
      - Initialization loop variable
      - Condition
      - Increment/Decrement loop variable

    Types of loops
    1. 'while' loop *
      - Syntax:
        // initialization
        while (condition) {
          // loop logic
          // increment/decrement loop variable
        }
*/
