/*
  # Scope: Space where the variable can be accessed
   - Global Scope: Variable can be accessed anywhere in the file
   - Function Scope: Variable can only be accessed within the function
   - Block Scope ({}): Variable can only be accessed within the block

  # Declare variables
    - var, let, const
    - ES6: ECMAScript 6 [2015] (versioning for JavaScript)
      - let, const (Block Scope)

    - var: Variables
      - Can be re-declared
      - Can be re-assigned
      - Function Scope
    - let: Variables
      - Cannot be re-declared
      - Can be re-assigned
      - Block Scope
    - const: Constants
      - Cannot be re-declared
      - Cannot be re-assigned
      - Block Scope

  # Future:
    - Function Scope
*/
