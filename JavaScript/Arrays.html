/*
  # Arrays: Collection of multiple elements
    - Syntax: let array = [element1, element2, element3, ...]
    - Different data type elements can exist within the same array
    - Access elements using index
      - Syntax: arr[index]
        - Index: Position - 1
        - String of length 'n', index range: 0 -> n-1
        - Access non-existent index: undefined
      - Update element at a given index
        - Syntax: arr[index] = newValue
        - Append element to the end of the array
          - arr[arr.length] = newValue
      - Iterate an array (using loops concept)
        - Syntax:
          for (let i=0; i < arr.length, i++) {
            console.log(arr[i])
          }
      - Multi-dimensional array: Nested array
        - Example: arr1 = [[2, 4], [3, 5]]
      - Properties:
        - length: Total number of elements in the array
      - Methods
        - indexOf() **: Returns the index of the FIRST occurrence of the element in the array, else returns -1
        - lastIndexOf(): Returns the index of the LAST occurrence of the element in the array, else returns -1
        - includes() **: Returns true if the element exists, else returns false (case sensitive)
        - push(newElement) **: Add a new element to the end of the array
          - Will update the original array
          - Return the updated length of the array
        - pop() **: Remove the last element from the array
          - Will update the original array
          - Return the removed element
        - unshift(newElement): Add a new element to the start of the array
          - Will update the original array
          - Return the updated length of the array
        - shift(): Remove the first element of the the array
          - Will update the original array
          - Return the removed element
        - concat(): Merge two arrays
          - Will not update the original array
          - Returns the merged array
        - slice(startIndex, endIndex) **: Returns a sub array
        (Ensure startIndex < end Index)
          - startIndex, endIndex: startIndex to endIndex - 1
          - startIndex: startIndex to the end of the array
          - No arguments: Complete array
        - reverse(): Reverse the order of array elements
          - Will update the original array
          - Return the reversed array
        - join(): Joint the array elements to form a string (Array -> String)
          - Will not update the original array
          - Returns the joined string
          - Notes:
            - No arguments: Join elements separated by comma
            - Separator: Join elements separated by the separator

      - Additional Notes:
        - In JavaScript:
          - Strings are immutable
          - Arrays are mutable
        - String -> Array
          - split(): Split a string to form elements of an array
          - Arguments:
            - No arguments: Returns an array with complete string as one element
            - Empty string: Returns an array with individual characters as elements
            - Separator: Returns an array with strings as elements, which were split using the separator

*/
