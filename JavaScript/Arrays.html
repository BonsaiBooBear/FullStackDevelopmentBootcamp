/*
  # Arrays: Collection of multiple elements
    - Syntax: let array = [element1, element2, element3, ...]
    - Different data type elements can exist within the same array
    - Access elements using index
      - Syntax: arr[index]
        - Index: Position - 1
        - String of length 'n', index range: 0 -> n-1
        - Access non-existent index: undefined
      - Update element at a given index
        - Syntax: arr[index] = newValue
        - Append element to the end of the array
          - arr[arr.length] = newValue
*/
