/*
  # DOM (Document Object Model)
    - Refers to the object representation of the HTML 'document tree'
    - Reserved keyword: document
    - Nodes: Represent individual elements in the DOM tree

    - Properties:
      - body: Returns the body node of the HTML document
      - head: Returns the head node of the HTML document
      - title: Returns the title node of the HTML document
      (CANNOT directly access child nodes further than this)
      - children: Returns all child nodes of given node
      - firstElementChild: Returns the first child node of a given node
      - lastElementChild: Returns the first child node of a given node
      - nextElementSibling: Returns the next sibling element of a given node

    - Methods (DOM selectors)
      - getElementById('id') **: Returns the node with the specified 'id' value
      - getElementsByClassName('class') **: Returns an HTMLCollection (array of nodes) which has the specified 'class'
      - getElementsByTagName('tag') **: Returns an HTMLCollection (array of nodes) which use the specified 'tag' such as h1, body, p, etc.

    - DOM Content Manipulations (Access/Update)
      - innerText: Refers to the inner text content of the node
      - innerHTML: Refers to the inner HTML content of the node
      (May behave like innerText if there is only text content inside the node)
      - value: Refers to the data entered within the form field node

    - DOM Attribute Manipulation (Access/Update)
      - Syntax: node.attributeName
      - 'class' attribute:
        - node.classList: Refers to the collection of classes assigned to a specific node
          - node.classList.add('newClass'): Add a class
          - node.classList.remove('existingClass'): Remove a class

    - DOM Style Manipulations (Access/Update)
      - Single Style property
        - Syntax: node.style.propertyName (convert "-" properties to camelCase)
      - Multiple Style properties
        - Syntax: node.style.cssText = 'property1: value 1; property2: value2;' (no camelCase conversions)

    - DOM Node Manipulation (Access/Update)
      - document.createElement('tag'): Create a new node based on the HTML tag provided
      - node.appendChild(childNode): Append the child node to the specified node
      - node.append(childNode1, childNode2, ...): Appends multiple child notes to the specified node
      - node.remove(): Remove/Delete the specified node from the DOM


  # Additional Notes:
    - BOM (Browser Object Model)
      - Reserved keyword: window
*/
