/*
  # Array Higher Order Methods:
    - Methods which use callback Syntax

    1. forEach() ***
      - Syntax:
        arr.forEach((element, index) => {
          // Code which executes for each element
        })
      - Iterate through the elements of the array and execute callback for each element
      - Callback parameters:
        - 1st parameter: Element
        - 2nd parameter: Index
      - Does not return anything (undefined)
    2. filter() ***
      - Syntax:
        arr.filter((element, index) => {
          // return "condition"
        })
      - Iterate through the elements of the array and execute callback for each element
      - Callback parameters:
        - 1st parameter: Element
        - 2nd parameter: Index
      - Returns a filtered array based on the condition provided. For each element, a boolean value is returned by the callback.
        - If True: Value is added to the filtered array
        - If False: Value is NOT added to the filtered array (skipped)
      - Does NOT modify the original array
      - Additional pointers:
        - IF "true" boolean value is returned by the callback, the filtered array will have all elements
        - IF "false" boolean value is returned by the callback, the filtered array will be an empty array
        - filteredArray.length <= originalArray.length

  # Additional Notes:
    - When looking for beginning max and mins, use the Number object to find positive and negative infinities.
        - let max = Number.NEGATIVE_INFINITY (smallest number possible before looping through elements that could have really large negative numbers)
        - let min = Number.POSITIVE_INFINITY (largest number possible before looping through elements that could have really large positive numbers)
