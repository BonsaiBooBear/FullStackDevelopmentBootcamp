/*
  # Array Higher Order Methods:
    - Methods which use callback Syntax

    1. forEach() ***
      - Syntax:
        arr.forEach((element, index) => {
          // Code which executes for each element
        })
      - Iterate through the elements of the array and execute callback for each element
      - Callback parameters:
        - 1st parameter: Element
        - 2nd parameter: Index
      - Does not return anything (undefined)
    2. filter() ***
      - Syntax:
        arr.filter((element, index) => {
          // return "condition"
        })
      - Iterate through the elements of the array and execute callback for each element
      - Callback parameters:
        - 1st parameter: Element
        - 2nd parameter: Index
      - Returns a filtered array based on the condition provided. For each element, a boolean value is returned by the callback.
        - If True: Value is added to the filtered array
        - If False: Value is NOT added to the filtered array (skipped)
      - Does NOT modify the original array
      - Additional pointers:
        - IF "true" boolean value is returned by the callback, the filtered array will have all elements
        - IF "false" boolean value is returned by the callback, the filtered array will be an empty array
        - filteredArray.length <= originalArray.length
    3. map() ***
      - Syntax:
        arr.map((element, index) => {
          // return [modifiedElement]
        })
      - Iterate through the elements of the array and execute callback for each element
      - Callback parameters:
        - 1st parameter: Element
        - 2nd parameter: Index
      - Returns a modified array based on the return value provided. For each element, a modified value can be returned, which will be the element at that index in the returned array.
      - Does NOT modify the original array
      - Additional pointers:
        - mappedArray.length = originalArray.length
        - If nothing is returned in the function, undefined is the value stored at that index.
    4. some()
      - Syntax:
        arr.some((element, index) => {
          // return "condition"
        })
      - Iterate through the elements of the array and execute callback for each element
      - Callback parameters:
        - 1st parameter: Element
        - 2nd parameter: Index
      - Returns a true/false based on the condition provided, if there exists "at least one element" that satisfies the condition.
      - Similar to OR (||)
    5. every()
      - Syntax:
        arr.every((element, index) => {
          // return "condition"
        })
      - Iterate through the elements of the array and execute callback for each element
      - Callback parameters:
        - 1st parameter: Element
        - 2nd parameter: Index
      - Returns a true/false based on the condition provided, if "all the elements" satisfy the condition.
      - Similar to AND (&&)
    6. find()
      - Syntax:
        arr.find((element, index) => {
          // return "condition"
        })
      - Iterate through the elements of the array and execute callback for each element
      - Callback parameters:
        - 1st parameter: Element
        - 2nd parameter: Index
      - Returns the first element in an array that matches the condition provided. If none of the elements match the condition, then undefined is returned.
    7. findIndex()
      - Syntax:
        arr.findIndex((element, index) => {
          // return "condition"
        })
      - Iterate through the elements of the array and execute callback for each element
      - Callback parameters:
        - 1st parameter: Element
        - 2nd parameter: Index
      - Returns the index of the first element in an array that matches the condition provided. If none of the elements match the condition, then -1 is returned.

  # Additional Notes:
    - When looking for beginning max and mins, use the Number object to find positive and negative infinities.
        - let max = Number.NEGATIVE_INFINITY (smallest number possible before looping through elements that could have really large negative numbers)
        - let min = Number.POSITIVE_INFINITY (largest number possible before looping through elements that could have really large positive numbers)
