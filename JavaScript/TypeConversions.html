/*
  # Type Conversion: Converting a value from one data type to another
    - Implicit type conversion (Automatic)
      - a.k.a. Type coercion
      - Examples:
        - String to Number
          - Equality operator (==)
          - Arithmetic operator excluding the '+' operator
            - console.log('5'-5) // 0
            - console.log('5'+5) // 55 (concatenation)
        - Number to String
          - Arithmetic operator '+'
            - console.log(5 + '5' - 5) // '55' - 5 = 0
            - console.log(5 + '5' / 5 - 5) // 5 + '1' - 5 = '6' - 5 + 1
        - Any data type to Boolean (Truthy/Falsy)
          - Value is directly placed in the condition placeholder
    - Explicit type conversion (Manual)
      - a.k.a. Type casting
      - Examples:
        - Inbuilt constructors
          - Number(): Any data type to number
          - String(): Any data type to string
          - Boolean(): Any data type to boolean
            - console.log(Boolean(5)) // true
            - console.log(Boolean(-1)) // true
            - console.log(Boolean(0)) // false
            - console.log(Boolean(' ')) // true
            - console.log(Boolean('')) // false
            - console.log(Boolean(undefined)) // false
            - console.log(Boolean(null)) // false

  # NaN: Not a Number
    - Represents an invalid or un-representable number
      - console.log('Abdiel' - 5) // NaN
    - isNaN(): To check if a value is NaN or not
      - cannot compare directly to NaN, because it is not a valid number
      - Ex. Cannot do if(5 == NaN). Instead: if(isNaN(variable))
    - typeof NaN = Number (silly, but used to invalidate numbers)

  # Truth and Falsy values:
    - Truthy values: Any values tht converts to true when evaluated in a Boolean context
      - Number: Any non-zero number
      - String: Any non-empty string
    - Falsy values: Any values tht converts to false when evaluated in a Boolean context
      - Number: 0, NaN
      - String: Empty string ('')
      - undefined
      - null
*/
