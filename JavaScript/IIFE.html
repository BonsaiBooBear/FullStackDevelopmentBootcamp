/*
  # IIFE (Immediately Invoked Function Expression)
    - A.K.A "Self-Invoking Function"
    - A function which gets invoked during the declaration itself.
    - Syntax
      (function (){
        // code
      })(); (semi-colon needed for multiple anonymous functions)
    - Use cases:
      - Avoid Global Scope pollution (Mimic Block Scope)
      - Achieve modularization

*/
