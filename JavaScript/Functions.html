/*
  # Functions:
    - A set/block of code which is intended to perform a particular task
    - Function can be executed as many times as needed
    - Advantages of functions:
      - Avoid repeated code (redundancy)
        - DRY (Don't Repeat Yourself) Principle
      - Modular code (Organized code)

    - For any function to be executed in JS, it needs two steps:
      - Declaration
      - Invocation (Calling the function)

    - Types of functions:
      - Standard function
      - Anonymous function
        - Does not have a name
        - Used as a function expression

    - Syntax:
      - Declaration
        - STANDARD FUNCTION
          function name () {
            // code
          }
        - ANONYMOUS FUNCTION
          const name = function () {
            // code
          }
      - Invocation
        name()

    - Parameters and arguments:
      - Can make a function dynamic (For different Input, you get different Output)
      - Parameters: Function variables (Values expected by the function)
      - Arguments: Actual values passed for function parameters uring invocation
      - Syntax:
          - Declaration:
            function func1(a, b) { // Parameters
              // code
            }
          - Invocation:
            func1(a, b) // Arguments
    
    - 'return' keyword:
      - Return a value at the place where the function was invoked
      - Terminates the function execution at the point where the return keyword was used

*/
