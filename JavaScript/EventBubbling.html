/*
  #  Event Propagation: Process by which events are handled and propagated through the DOM tree (Applicable for nested elements)
    - 2 Phases
      - Even Capturing: Event is propagated from last ancestor to the child node, and so on, until the target element.
      - Event Bubbling ***: Even is propagated (spread) from target node to the parent node, and so on, until the last ancestor element.

    - Stop Event Propagation
      - event.stopPropagation()
*/
