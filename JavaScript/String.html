/*
  # Strings
    - "", '', ``
    - Escape character (\)
    - Next line character (\n)
    - Tab space character (\t)

    - Access a specific character in a string using index
      - Syntax: str[index]
        - Index: Position - 1
        - String of length 'n', index range: 0 -> n-1

    - '+' operator works as a concatenation operator with strings
      - String + String = String
      - String + Any data = String
      - Any data + String = String

    - Properties:
      - length: Total number of characters in a string (spaces included)
        - Empty strings do not have a length
    - Methods:
      - toUpperCase(): Return the string in upper case
      - toLowerCase(): Return the string in lower case
      - indexOf() **: Returns the index of the FIRST occurrence of the substring
        (any portion of the string) in the string, else returns -1
      - lastIndexOf(): Returns the index of the LAST occurrence of the substring
        (any portion of the string) in the string, else returns -1
      - includes() **: Returns true if the substring exists, else returns false (case sensitive)
      - concat(): Concatenate two or more strings (var1.concat(var2), var1.concat(var2, var3))
        - Alternative: + operator
      - charAt(index): Returns the character at the specified index
        - Alternative: str[index]
      - slice (startIndex, endIndex) **: Returns a substring
        (Ensure startIndex < end Index)
        - startIndex, endIndex: startIndex to endIndex - 1
        - startIndex: startIndex to the end of the string
        - No arguments: Complete string
        - Alternatives:
          - substr(startIndex, substringLength): substringLength starting from startIndex
          - substring(startIndex, endIndex): Similar to slice
            - negative index will be treated as 0
            - startIndex > endIndex: index will be swapped
      - replace(substring, replaceValue): Return a string where substrings are
        replaced with replaceValue
          - replaceAll(substring, replaceValue): Replace all occurrences
*/
