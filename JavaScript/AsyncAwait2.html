/*
  # Async Await
    - Syntactic Sugar / Alternative for promise handling
    - 'async' keyword: Used to make any function asynchronous
    - 'await' keyword:
        - Used to pause the execution of an async function until the promise is resolved
        - Returns the resolvedResult of the promise when the promise is resolved
        - Throws the rejected result of the promise when the promise is rejected
    - Syntax:
      async function handlePromise() {
        try {
          const resolvedResult = await promise1
          console.log(resolvedResult)
        } catch (rejectedResult) {
          console.log(rejectedResult)
        }
      }

    - Different type of asynchronous functions
      - async function abc() {}
      - const abc = async function() {}
      - const abc = async () => {}

*/
