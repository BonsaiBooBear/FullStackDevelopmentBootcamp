/*
  # Objects: Collections of key value pairs
    - keys (string)
    - values (any data type)
    - Syntax:
      let obj1 = {
        key1: value1,
        key2: value2,
        key3: value3,
        ,,,
      }

    - Access values using keys
      - Syntax 1: obj1['key1']
      - Syntax 2: obj1.key1 **

    - Add/Update values using keys
      - Syntax 1: obj1['key1'] = value1
      - Syntax 2: obj1.key1 = value1 **

    - Delete a key
      - 'delete' keyword
      - Syntax: obj1.key1

    - Properties and Methods (keys)
      - Properties: Keys holding any value other than function
      - Methods: Keys holding functions as a reference

    - Inbuild object methods:
      - Object.keys(obj1): Returns an array of keys belonging to a specific object
      - Object.values(obj1): Returns an array of values belonging to a specific object
      - Object.entries(obj1): Returns an array of [key, value] pairs belonging to a specific object

    - JSON: JavaScript Object Notation
      - JSON.stringify(): Convert a JS object to JSON string
      - JSON.parse(): Convert a JSON string to a JS object

    - Side note:
      - [object Object]: Object in the HTML document flow

  # Constructor: Create objects of similar type
    - Create a new object from constructor: 'new' keyword
    - Inbuilt constructors: Used by JS behind the scenes
      - Number()
      - String()
      - Boolean()

    - typeof null = object (similar to NaN being a number, they had to categorize null somewhere)
*/
