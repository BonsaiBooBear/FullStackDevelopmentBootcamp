/*
  # Asynchronous JavaScript
    - Synchronous code: Executed one line at a time
    - Asynchronous code: Executed as an independent logic (Executed parallelly)

    - JavaScript is a single threaded language
      - There is only one thread (main thread) which executes the complete JS code one expression at a time.
      - Still, it can execute asynchronous code with the help of other elements in the JS Engine (Event Loop, Task Queues, Web APIs).

    # Timer methods:
      - setTimeout(): Execute code after a specified time
        - Syntax:
          setTimeout(()=> {
            //logic
          }, time) // (Callback, time)
          - Parameters:
            - Callback: Function to execute after the specified time
            - time: in milliseconds (1000ms = 1s)
          - Return:
            - timerId: Unique ID assigned to the timeout created by setTimeout method
        - clearTimeout(timerId): Clear the timeout created by setTimeout method
      - setInterval(): Execute code repeatedly after a specified time interval
          - Syntax:
          setInterval(()=> {
            //logic
          }, time) // (Callback, time)
          - Parameters:
            - Callback: Function to execute after the specified time repeatedly
            - time: in milliseconds (1000ms = 1s)
          - Return:
            - timerId: Unique ID assigned to the interval created by setInterval method
        - clearInterval(timerId): Clear the interval created by setInterval method

*/
