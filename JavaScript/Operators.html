  # Operators: Perform logical operations
    - Arithmetic Operators **
      - Addition (+)
      - Subtraction (-)
      - Multiplication (*)
      - Division (/): Results in Quotient
      - Modulus (%): Results in Remainder
      - Increment (++)
        - Pre-increment (++x): Increment, then access
        - Post-increment (x++): Access, then increment
      - Decrement (--)
        - Pre-decrement (--x): Decrement, then access
        - Post-decrement (x--): Access, then decrement
      - Exponential (**)
    - Assignment Operators
      - Assignment (=)
      - Addition Assignment (+=)
      - Subtraction Assignment (-=)
      - Multiplication Assignment (*=)
      - Division Assignment (/=)
      - Modulus Assignment (%=)
      - Exponential Assignment (**=)
    - Comparison Operators ** (Return boolean: true/false)
      - Greater than (>)
      - Lesser than (<)
      - Greater than or equal to (>=)
      - Lesser than or equal to (<=)
      - Equality Operator (==): Equal value, NOT data type
      - Identity Operator (===): Equal value AND data type (Strict equality operator)
      - Inequality Operator (!=): Non-Equal value
      - Non-Identity Operator (!==): Non-equal value OR data type
    - Logical Operators ** (Return boolean: true/false)
      - Logical AND (&&): ALL TRUE -> TRUE
      - Logical OR (||): ONE TRUE -> TRUE
      - Logical NOT (!): TRUE -> TRUE, FALSE -> TRUE
